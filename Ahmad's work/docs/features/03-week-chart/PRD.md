# Week Ahead Chart - PRD

## Goal
A dashboard tile that plots three load metrics (tasks, routines, packing items) for the current day + next 6 days, so users can see their week's intensity at a glance.

## Success
- Renders a single SVG line chart with three colour-coded lines and dot markers per day.
- Today is highlighted on the X-axis label.
- The chart updates as the selected date changes (the 7-day window starts at the selected date).

## Out of Scope
- Drill-down on hover.
- Filtering metrics in/out.
