# DECISIONS: Dashboard Command Center

## D1: Dashboard Is the Product

The dashboard gets the most polish because it carries the 60-second demo narrative.

## D2: Server Data, Client Interactions

Initial plan data should load server-side where practical. Interactive and animated components stay client-side.

## D3: Bottleneck Is Not Hidden

The main bottleneck appears in the header and is reinforced visually in the Strategy Map.
