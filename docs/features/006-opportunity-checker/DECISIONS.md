# DECISIONS: Opportunity Checker

## D1: Embedded in Dashboard

The checker belongs on the dashboard because the current strategy context should be visible when the student evaluates an opportunity.

## D2: Must Be Willing to Say No

The prompt must force an opinionated recommendation. A hedged answer weakens the product.

## D3: Yes Requires a Cut

If the recommendation is yes or yes with conditions, the result must explain what gets cut, capped, or deferred.
