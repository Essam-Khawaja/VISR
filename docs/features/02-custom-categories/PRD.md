# Custom Categories - PRD

## Goal
Allow users to add their own event categories (e.g. "Volunteering", "Music practice") that show up everywhere built-ins do.

## Success
- A user can add a custom category in Settings.
- Custom categories appear in the EventForm picker and the "Pack defaults per category" manager.
- Events using a custom category render with a recognisable generic styling on the timeline / week / notes.

## Out of Scope
- Per-category icon picker (uses a generic `Tag` icon).
- Custom palette per category.
