# DECISIONS: Landing and Onboarding

## D1: Multi-Step Form

Use a multi-step form instead of a single long form because the student is already overwhelmed. The flow should progressively narrow from goal to context to messy details.

## D2: Brain Dump Is Required

The brain dump is required because it carries the highest-signal context for Claude. It should accept messy, unstructured text.

## D3: Demo Escape Hatch

If generation fails, the UI must offer the demo route. No user should hit a dead end during judging.
