# Cursor Prompt: Implement a Feature

Use this prompt when coding a feature:

```md
Implement this feature:

`docs/features/[feature-folder]/`

Before coding:
1. Read the global architecture docs.
2. Read all docs in the feature folder.
3. Summarize the feature goal.
4. List the files you plan to modify.
5. Implement only the next unchecked task in the feature `TASKS.md`.

After coding:
1. Update the feature task list.
2. Add decisions to feature `DECISIONS.md` if needed.
3. Tell me how to validate the change.
```
